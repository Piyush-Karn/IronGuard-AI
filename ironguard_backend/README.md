## рҹҡҖ Getting Started

### Prerequisites

| Requirement | Notes |
|------------|-------|
| рҹҗі Docker + Docker Compose | Recommended deployment method |
| рҹ”‘ Gemini API Key | Primary LLM provider |
| рҹ”‘ Mistral API Key | Fallback LLM provider |

### 1 вҖ” Configure your environment

Create a `.env` file inside `ironguard_backend/`:

```env
GEMINI_API_KEY=your_gemini_key
MISTRAL_API_KEY=your_mistral_key
```

### 2 вҖ” Start the system

```bash
docker compose up --build -d
```

### 3 вҖ” Initialize the threat database

```bash
docker compose exec backend python datasets/init_dataset.py
```

### 4 вҖ” Access the interfaces

| Interface | URL |
|-----------|-----|
| рҹ–ҘпёҸ Admin Dashboard | `http://localhost:5173` |
| рҹ“ҡ Swagger API Docs | `http://localhost:8000/docs` |
